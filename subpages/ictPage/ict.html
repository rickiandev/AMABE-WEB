ICT

Here with the ICT strand, students will learn the things needed for a future IT career related to Animation, Visual Graphics, and Computer Programming for Web Development and Game Development.

Animation
Learn how to draw, animate, create sound effects and a short animation video with Adobe Animate and Illustrate.
Visual Graphics
Learn how to Design different marketing, professional, and educational designs like Flyers, Brochure, CV, and Resume.
Computer Programming
Learn to create Games with Java or Python and How to create websites with HTML, CSS, and JavaScript.
